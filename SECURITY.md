# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in KrillClaw, please report it responsibly.

**Email:** hello@krillclaw.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to provide a fix within 7 days for critical issues.

**Please do not open a public GitHub issue for security vulnerabilities.**

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Scope

This policy applies to the KrillClaw runtime, build system, and official tooling.
